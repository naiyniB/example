Figma 毛玻璃效果实现

浅色毛玻璃效果

这是标准的浅色毛玻璃效果，类似于 iOS 和 macOS 的设计风格。

背景: rgba(255, 255, 255, 0.25)

模糊: 10px

边框: 1px rgba(255, 255, 255, 0.18)

深色毛玻璃效果

深色毛玻璃效果，适合在明亮背景上使用，提供更好的对比度。

背景: rgba(0, 0, 0, 0.2)

模糊: 15px

边框: 1px rgba(255, 255, 255, 0.1)

彩色毛玻璃效果

带有色调的毛玻璃效果，可以匹配品牌色彩或界面主题。

背景: rgba(147, 112, 219, 0.2)

模糊: 12px

边框: 1px rgba(147, 112, 219, 0.3)

Tailwind CSS 实现

使用 Tailwind 的 backdrop-blur 和透明度工具类实现。

类名: bg-white/20 backdrop-blur-lg

边框: border border-white/30

圆角: rounded-2xl

如何实现毛玻璃效果

CSS 属性

  • backdrop-filter: blur() - 核心模糊效果
  • background: rgba() - 半透明背景
  • border - 细微边框增强效果
  • box-shadow - 可选，增加深度

最佳实践

  • 确保背后有内容才能看到效果
  • 调整透明度平衡美观和可读性
  • 在移动设备上测试性能
  • 提供备用样式以防不支持
